# Texture Features Guide

Οδηγός για την ερμηνεία των texture features που υπολογίζει το `texture_features.py`

## 📊 Τι είναι τα Texture Features;

Τα texture features περιγράφουν την **υφή** (texture) μιας εικόνας - δηλαδή το πώς κατανέμονται τα χρώματα και οι εντάσεις χωρικά.

## 🔍 Κατηγορίες Features

### 1. GLCM Features (Gray-Level Co-occurrence Matrix)

Μετρούν πόσο συχνά εμφανίζονται ζεύγη pixels με συγκεκριμένες εντάσεις σε συγκεκριμένες αποστάσεις/γωνίες.

#### **Contrast** (0 - ∞)
- Μετρά την τοπική variation στην εικόνα
- **Υψηλό** (>100): Πολλές απότομες αλλαγές, έντονα edges, "τραχιά" υφή
- **Χαμηλό** (<50): Ομαλή υφή, λίγες αλλαγές
- **Παράδειγμα**: Τούβλα = υψηλό, Γαλάζιος ουρανός = χαμηλό

#### **Homogeneity** (0 - 1)
- Μετρά την ομοιογένεια της εικόνας
- **Υψηλό** (>0.8): Πολύ ομοιόμορφη υφή
- **Χαμηλό** (<0.5): Ετερογενής υφή
- **Σημείωση**: Αντίθετο του Contrast

#### **Energy** (0 - 1)
- Μετρά την ομοιομορφία/επαναληπτικότητα
- **Υψηλό** (>0.5): Επαναλαμβανόμενα patterns
- **Χαμηλό** (<0.2): Τυχαία κατανομή
- **Παράδειγμα**: Καρό μοτίβο = υψηλό, Τυχαίος θόρυβος = χαμηλό

#### **Correlation** (-1 - 1)
- Μετρά τη γραμμική εξάρτηση γειτονικών pixels
- **Υψηλό** (>0.8): Pixels κοντά τους έχουν παρόμοιες τιμές
- **Χαμηλό** (<0.3): Ανεξάρτητα pixels

#### **Dissimilarity** (0 - ∞)
- Παρόμοιο με Contrast αλλά πιο ευαίσθητο σε μικρές διαφορές

#### **ASM** (Angular Second Moment) (0 - 1)
- Το τετράγωνο του Energy
- Μετρά την "τάξη" στην εικόνα

---

### 2. LBP Features (Local Binary Patterns)

Περιγράφει την τοπική υφή γύρω από κάθε pixel συγκρίνοντάς το με τους γείτονές του.

#### **Histogram**
- Κατανομή των LBP codes
- Χρήσιμο για pattern recognition

#### **Mean, Std**
- Στατιστικά του LBP
- Υψηλό std = πολυποίκιλη υφή

---

### 3. Edge Features

Μετρούν την παρουσία edges (άκρων) στην εικόνα.

#### **Edge Magnitude Mean** (0 - ∞)
- Μέση ένταση των edges
- **Υψηλό** (>50): Πολλά/έντονα edges (π.χ. κτίρια, γραμμές)
- **Χαμηλό** (<20): Λίγα edges (π.χ. ουρανός, θάλασσα)

#### **Edge Density** (0 - 1)
- Ποσοστό pixels που είναι edges
- **Υψηλό** (>0.3): Πολύπλοκη εικόνα με πολλά edges
- **Χαμηλό** (<0.1): Απλή εικόνα με λίγα edges

#### **Horizontal/Vertical Edges**
- Κατεύθυνση των edges
- Χρήσιμο για ανίχνευση οριζόντιων/κάθετων δομών

---

### 4. Statistical Features

Βασικά στατιστικά μέτρα της κατανομής εντάσεων.

#### **Mean** (0 - 255)
- Μέση ένταση (brightness)

#### **Std** (Standard Deviation) (0 - ~127)
- Πόσο ποικίλλουν οι εντάσεις
- **Υψηλό** (>50): Μεγάλη ποικιλία (από σκούρα σε φωτεινά)
- **Χαμηλό** (<20): Ομοιόμορφη ένταση

#### **Entropy** (0 - 8)
- Μετρά την "αταξία" ή πολυπλοκότητα
- **Υψηλό** (>6): Πολύ τυχαία/πολύπλοκη εικόνα
- **Χαμηλό** (<3): Απλή/προβλέψιμη εικόνα

#### **Skewness**
- Ασυμμετρία της κατανομής
- **Θετικό**: Περισσότερα σκούρα pixels
- **Αρνητικό**: Περισσότερα φωτεινά pixels

#### **Kurtosis**
- "Μυτερότητα" της κατανομής
- **Υψηλό**: Πολλά pixels σε ακραίες τιμές
- **Χαμηλό**: Ομοιόμορφη κατανομή

## 🎯 Χρήση για Matching Tiles

### Παρόμοια Texture → Πιθανώς γειτονικά tiles

Δύο tiles με παρόμοια texture features μπορεί να:
- Ανήκουν στην ίδια περιοχή (π.χ. ουρανός, γρασίδι)
- Είναι γειτονικά tiles
- Έχουν παρόμοια υφή (χρήσιμο για matching)

### Συνδυασμός με Color Histograms

- **Color Histogram**: Τι χρώματα υπάρχουν
- **Texture Features**: Πώς κατανέμονται τα χρώματα (υφή)

**Παράδειγμα:**
- Δύο tiles μπορεί να έχουν ίδιο color histogram (ίδια χρώματα)
- Αλλά διαφορετικά texture features (διαφορετική υφή)
→ Δεν είναι γειτονικά!

## 📖 Παραδείγματα

### Παράδειγμα 1: Γρασίδι
```json
{
  "glcm": {
    "contrast": 45.2,        // Μέτριο (μικρές αλλαγές)
    "homogeneity": 0.78,     // Υψηλό (ομοιόμορφο)
    "energy": 0.12,          // Χαμηλό (όχι επαναληπτικό)
    "correlation": 0.85      // Υψηλό (ομαλό)
  },
  "edges": {
    "edge_density": 0.08     // Χαμηλό (λίγα edges)
  },
  "statistical": {
    "entropy": 5.2,          // Μέτριο
    "std": 35.6              // Μέτριο variation
  }
}
```

### Παράδειγμα 2: Τοίχος από Τούβλα
```json
{
  "glcm": {
    "contrast": 180.5,       // Πολύ υψηλό (έντονες αλλαγές)
    "homogeneity": 0.45,     // Χαμηλό (ετερογενές)
    "energy": 0.35,          // Μέτριο (κάποια patterns)
    "correlation": 0.65      // Μέτριο
  },
  "edges": {
    "edge_density": 0.42     // Υψηλό (πολλά edges από αρμούς)
  },
  "statistical": {
    "entropy": 6.8,          // Υψηλό (πολύπλοκο)
    "std": 62.3              // Υψηλό variation
  }
}
```

### Παράδειγμα 3: Καθαρός Ουρανός
```json
{
  "glcm": {
    "contrast": 12.3,        // Πολύ χαμηλό (σχεδόν καμία αλλαγή)
    "homogeneity": 0.95,     // Πολύ υψηλό (πολύ ομοιόμορφο)
    "energy": 0.08,          // Χαμηλό
    "correlation": 0.92      // Πολύ υψηλό (πολύ ομαλό)
  },
  "edges": {
    "edge_density": 0.02     // Πολύ χαμηλό (σχεδόν καθόλου edges)
  },
  "statistical": {
    "entropy": 3.1,          // Χαμηλό (απλό)
    "std": 15.2              // Χαμηλό variation
  }
}
```

## 💡 Tips για Matching

1. **Ομοιόμορφες περιοχές** (ουρανός, τοίχος):
   - Χρησιμοποίησε κυρίως **homogeneity** και **entropy**

2. **Υφές με patterns** (καρό, τούβλα):
   - Χρησιμοποίησε **energy** και **LBP histogram**

3. **Περιοχές με edges** (κτίρια):
   - Χρησιμοποίησε **edge features** και **contrast**

4. **Συνδυασμός:**
   - Χρησιμοποίησε `compare_texture_features()` για συνολική σύγκριση
